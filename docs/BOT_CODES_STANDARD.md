# CITARION Стандарт Кодов Ботов

## 0. Мета-Боты

Системные боты для координации и агрегации сигналов.

| Код | Имя | Стратегия | Описание |
|-----|-----|-----------|----------|
| LOGOS | Агрегатор | Signal Aggregation | Агрегация сигналов от всех ботов с оценкой качества |

## 1. Операционные Боты

Базовые торговые боты для стандартных стратегий.

| Код | Имя | Стратегия | Описание |
|-----|-----|-----------|----------|
| MESH | Сетка | Grid Trading | Размещает ордера на покупку/продажу через фиксированные интервалы |
| SCALE | Усреднение | Dollar Cost Averaging | Усредняет цену входа через несколько ордеров |
| BAND | Полосы | Bollinger Bands | Торговля по сигналам полос Боллинджера |

## 2. Институциональные Боты

Продвинутые стратегии для профессиональной торговли.

| Код | Имя | Стратегия | Описание |
|-----|-----|-----------|----------|
| PR | Спектр | Pairs Trading | Торговля коррелированными парами с возвратом к среднему |
| STA | Рид | Statistical Arbitrage | Использование статистических ценовых неэффективностей |
| MM | Архитектор | Market Making | Обеспечение ликвидности с захватом спреда bid/ask |
| MR | Эквилибрист | Mean Reversion | Торговля возврата цены к статистическому среднему |
| TRF | Крон | Trend Following | Следование за установленными рыночными трендами |

## 3. Аналитические Боты

Боты для анализа и генерации сигналов.

| Код | Имя | Стратегия | Описание |
|-----|-----|-----------|----------|
| PND | Аргус | Pump & Dump Detection | Обнаружение необычного объёма и движений цены |
| TRND | Орион | Trend Analysis | Мульти-таймфреймный анализ и торговля трендов |
| FCST | Видение | Price Forecasting | ML-прогнозирование цены |
| RNG | Диапазон | Range Trading | Торговля в определённых границах цены |
| LMB | Люми | AI Assistant | ИИ-помощник для торговли |
| WOLF | Волк | Technical Analysis | Продвинутый технический анализ и паттерны |

## 4. Частотные Боты

Высокоскоростные торговые боты по частоте исполнения.

| Код | Имя | Частота | Описание |
|-----|-----|---------|----------|
| ПАНЕЛЬ | Все + LOGOS | Мульти-частотная | Единая панель для всех частотных ботов с LOGOS |
| HFT | Гелиос | Ультра-высокая (<10мс) | Высокочастотная торговля с задержкой менее 10мс |
| MFT | Селена | Высокая (<100мс) | Среднечастотная торговля до 100мс |
| LFT | Атлас | Средняя (<1с) | Низкочастотная торговля до 1 секунды |

## 5. Другие системные боты

| Код | Имя | Функция |
|-----|-----|---------|
| ORA | Оракул | Чат UI + ИИ уведомления |

## 6. Интеграционные Боты

Внешние сигналы и копитрейдинг.

| Код | Имя | Функция |
|-----|-----|---------|
| SIG | Signal Bot | Парсинг сигналов TradingView и исполнение |
| CPY | CopyTrader | Копитрейдинг от мастер-трейдеров |

## 7. Организация Сайдбара

Сайдбар CITARION организован с вложенными подразделами:

```
▼ Боты (кликабельный раскрывающийся раздел)
  ▼ Мета
    └── LOGOS (Агрегатор)
  ▼ Операционные
    ├── MESH (Сетка)
    ├── SCALE (Усреднение)
    └── BAND (Полосы)
  ▼ Институциональные
    ├── PR (Спектр)
    ├── STA (Рид)
    ├── MM (Архитектор)
    ├── MR (Эквилибрист)
    └── TRF (Крон)
  ▼ Аналитические
    ├── PND (Аргус)
    ├── TRND (Орион)
    ├── FCST (Видение)
    ├── RNG (Диапазон)
    ├── LMB (Люми)
    └── WOLF (Волк)
  ▼ Частотные
    ├── ПАНЕЛЬ (Все + LOGOS)
    ├── HFT (Гелиос)
    ├── MFT (Селена)
    └── LFT (Атлас)
```

**Особенности:**
- Раздел "Боты" сворачиваемый (клик для раскрытия/сворачивания)
- Каждая категория сворачиваемая внутри раздела "Боты"
- Категория "Мета" на первом месте (содержит LOGOS)
- ПАНЕЛЬ на первом месте в Частотных
- Визуальные отступы показывают иерархию
- Активный бот/категория подсвечивается

## 8. Формат отображения в UI

```
[ИмяБота] (КОД) Описание - Статус
```

Пример: `[Гелиос] (HFT) Высокочастотная торговля - Активен`

## 10. Новые компоненты (Stage 2)

### 10.1 Genetic Algorithm Framework
**Файл:** `src/lib/genetic-v2/framework.ts`
- `GeneticOptimizer` — главный класс оптимизатора
- `GeneticOperators` — операторы мутации, кроссовера
- `FitnessCalculator` — расчёт fitness function
- `BOT_GENE_TEMPLATES` — шаблоны генов для ботов

### 10.2 Extended Signal Classifier
**Файл:** `src/lib/ml-v2/extended-classifier.ts`
- `ExtendedSignalClassifier` — ансамбль классификаторов
- `KNNClassifier` — k-NN с Lorentzian distance
- `FeatureExtractor` — извлечение 15+ фич

### 10.3 Trading Journal
**Файл:** `src/lib/logos-v2/trading-journal.ts`
- `TradingJournal` — главный класс журнала
- `JournalAnalyzer` — анализ паттернов
- `TradeAnalyzer` — AI-анализ сделок

### 10.4 Online Learning
**Файл:** `src/lib/vision-v2/online-learner.ts`
- `OnlineLearner` — инкрементальное обучение
- `MultiHorizonForecaster` — мульти-горизонт прогнозы
- `DriftDetector` — детекция concept drift
