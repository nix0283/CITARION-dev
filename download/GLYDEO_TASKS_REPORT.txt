│      - Предупреждения о ликвидации                                          │
│      - Автоматическое обновление статусов позиций и сигналов                │

│      - Уведомления в Telegram                                               │
│      - Уведомления в UI через SSE                                           │
│      - События: POSITION_OPENED, ORDER_FILLED, TP_HIT, SL_HIT, etc.         │
│                                                                              │
│   4. /api/notifications - SSE endpoint для real-time уведомлений:           │
│      - Server-Sent Events для UI                                            │
